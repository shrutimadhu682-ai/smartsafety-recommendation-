# 🛡️ Smart Safety Recommendation System  
### AI-Powered Safety Navigation, SOS Alerts & Real-Time Risk Detection

The **Smart Safety Recommendation System** is a safety-focused web application designed to help users — especially women — navigate safely, receive real-time risk alerts, send instant SOS signals, and stay connected with trusted guardians.

This project uses **Node.js**, **JavaScript**, **real-time location tracking**, and **risk analysis** to provide a secure navigation experience.

---

## 🚀 Features

### 🔹 **Safe vs Fast Route Recommendation**
- Shows both **fastest route** and **safest route**
- Uses risk scoring to highlight safer paths

### 🔹 **Fear Score / Risk Rating**
- Calculates risk based on area, time, incidents, and crowd presence
- Updates dynamically as the user moves

### 🔹 **Safety Heatmap**
- Color-coded map showing safe, moderate, and unsafe zones

### 🔹 **Real-Time Risk Alerts**
- Sends instant alerts if user enters a risky zone

### 🔹 **One-Tap SOS Button**
- Instantly alerts emergency contacts
- Sends live location automatically

### 🔹 **Voice-Activated SOS**
- Say a keyword (“Help”, “SOS”) to trigger alerts without touching the phone

### 🔹 **Guardian Tracking**
- Trusted contacts can follow the user’s real-time path

### 🔹 **Offline / Dead-Zone Alerts**
- Sends alerts if network/GPS is unavailable or user goes missing

### 🔹 **Decoy Mode (Hidden Interface)**
- App switches to a fake harmless screen in danger situations

### 🔹 **Incident Reporting**
- Users can report harassment, crime, or unsafe events
- Data feeds into risk mapping

## ⚙️ Installation & Setup

### **1. Clone the repository**
```bash
git clone https://github.com/shrutimadhu682-ai/smartsafety-recommendation-.git
cd smartsafety-recommendation-
```

### **2. Install dependencies**
```bash
npm install
```

### **3. Start the server**
```bash
npm start
```

### App runs on:
```
http://localhost:3000
```
